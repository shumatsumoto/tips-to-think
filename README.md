## :pencil2: Overview

I'm using Rails6 and Docker.
I created a simple Blog application.

## :hammer: Requirement

- macOS
- Docker 20.10.5
- Rails 6.1.3.1
- Ruby 2.7.1
- MySql 5.7
- Material Design for Bootstrap4

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/tips-to-think.git
```
Move to directory
```
$ cd tips-to-think
```
Rail server & Mysql server start
```
$ docker-compose up -d
$ （docker-compose build）
$ docker-compose ps
```
Access URL： 
http://localhost:3000

## :railway_car: Features

- Login / Logout / SignUp
